# Notes

Notas personales alojadas en las páginas de github mediante el uso de la wiki no lineal TiddlyWiki.

Puesto que la wiki se auto-guarda generando nuevo commits en el repositorio donde se aloja para clonar este último se recomienda usar el siguiente comando:

```shell
git clone --depth=1 https://github.com/benabhi/notes
```

Con la finalidad traer solo los ultimos cambios y ahorrar el tiempo que tardaria git en hacer un pull de la carpeta completa (que ya con los 100+ commits que tiene se demora un poco).
